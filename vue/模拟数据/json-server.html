# 全局安装

yarn add global json-server //使用yarn安装后无法运行使用命令，不知道为啥，使用npm可以

npm install -g json-server

# 使用

json-server --watch --port 3003 --host 127.1.1.1 db.json

--watch 表示启动服务，监听数据更改 --port 表示启动端口 --host 表示启动主机域名 db.json 表示生成的文件
